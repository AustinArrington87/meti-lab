What METI Lab does

  • Parses GeoJSON, KML, Shapefile, GPX, and GML files
  • Accepts a whole shapefile folder — just drag it in
  • Fixes invalid polygons (self-intersections, winding order)
  • Guides you through required METI fields via chat
  • Checks for spatial conflicts with existing METI sources
  • Exports a valid METI GeoJSON ready for the source ledger
